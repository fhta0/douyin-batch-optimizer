# 抖店批量属性优化助手 - 安装指南

## 📦 插件文件清单

```
douyin-batch-optimizer/
├── manifest.json          # 插件配置文件
├── content.js             # 核心逻辑脚本
├── popup.html             # 弹窗界面
├── popup.js               # 弹窗逻辑
├── generate-icons.html    # 图标生成器（HTML页面）
├── icon16.png             # 16x16图标（需要生成）
├── icon48.png             # 48x48图标（需要生成）
├── icon128.png            # 128x128图标（需要生成）
├── README.md              # 说明文档
└── INSTALL.md             # 本安装指南
```

## 🔧 安装步骤

### 步骤1：生成图标文件

由于无法直接生成PNG文件，你需要手动生成图标：

**方法一：使用HTML页面生成（推荐）**

1. 在浏览器中打开 `generate-icons.html` 文件
2. 点击"生成图标"按钮
3. 右键点击每个图标，选择"图片另存为..."
4. 保存到 `douyin-batch-optimizer` 文件夹，文件名分别为：
   - `icon16.png`
   - `icon48.png`
   - `icon128.png`

**方法二：使用在线工具**

1. 访问在线SVG转PNG工具（如：https://convertio.co/zh/svg-png/）
2. 上传 `icon.svg` 文件
3. 转换为16x16、48x48、128x128三种尺寸
4. 分别保存为 `icon16.png`、`icon48.png`、`icon128.png`

### 步骤2：安装Chrome插件

1. **打开Chrome扩展程序页面**
   - 在Chrome地址栏输入：`chrome://extensions/`
   - 按回车键访问

2. **开启开发者模式**
   - 在页面右上角找到"开发者模式"开关
   - 将其打开

3. **加载插件**
   - 点击"加载已解压的扩展程序"按钮
   - 选择 `douyin-batch-optimizer` 文件夹
   - 点击"选择文件夹"

4. **完成安装**
   - 插件会出现在扩展程序列表中
   - 建议点击插件旁的"固定"按钮，将图标固定到工具栏

## 🚀 使用方法

### 1. 打开抖店商品管理页面

在浏览器中访问：
```
https://fxg.jinritemai.com/ffa/g/list
```

确保页面显示"你有XXX个商品属性待优化"的提示。

### 2. 启动插件

1. 点击浏览器右上角的插件图标（紫色"抖"字图标）
2. 在弹窗中点击"开始优化"按钮
3. 插件会自动开始批量优化商品属性

### 3. 查看进度

- 弹窗中会显示实时进度：
  - 运行状态
  - 已处理批次
  - 已优化商品数
  - 待优化商品数
- 日志区域会显示详细操作记录

### 4. 停止优化

如需停止，点击"停止"按钮即可。

## ⚙️ 插件配置

如需修改配置，可以编辑 `content.js` 文件：

```javascript
const CONFIG = {
    interval: 3000,        // 每批处理间隔（毫秒）
    maxRetries: 3,         // 最大重试次数
    debug: true            // 调试模式
};
```

## ⚠️ 注意事项

1. **请确保当前页面为抖店商品管理页面**，否则插件无法正常工作
2. **插件会自动处理，无需人工干预**
3. **每批处理间隔3秒**，避免操作过快导致问题
4. **可以随时点击"停止"按钮暂停**
5. **建议在网络稳定的环境下使用**

## 🔍 常见问题

### Q: 插件安装后没有显示图标？
A: 点击Chrome工具栏的拼图图标，找到"抖店批量属性优化助手"，点击旁边的"固定"按钮。

### Q: 点击"开始优化"没有反应？
A: 请确保当前页面是抖店商品管理页面（URL包含fxg.jinritemai.com），并且页面已完全加载。

### Q: 优化过程中出现错误？
A: 插件会自动重试，如多次失败会自动停止。可以刷新页面后重新启动。

### Q: 可以同时优化多个店铺吗？
A: 不可以，插件一次只能处理一个店铺的商品。
