# 抖店批量属性优化助手

Chrome插件，用于自动批量优化抖店商品属性。

## 功能特点

- ✅ 自动批量优化商品属性
- ✅ 自动勾选协议
- ✅ 自动确认提交
- ✅ 循环处理直到完成
- ✅ 实时显示进度
- ✅ 可暂停/恢复
- ✅ 设置中心支持自定义参数

## 安装步骤

1. 打开Chrome浏览器，访问 `chrome://extensions/`
2. 开启右上角的"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择 `douyin-batch-optimizer` 文件夹
5. 安装完成！

## 使用方法

1. 在浏览器中打开抖店商品管理页面
   - URL: `https://fxg.jinritemai.com/ffa/g/list`

2. 点击浏览器右上角的插件图标

3. 点击"开始优化"按钮

4. 插件会自动：
   - 点击"批量优化属性"按钮
   - 勾选"我确认商品属性信息无误"协议
   - 点击"一键优化属性"按钮
   - 确认提交
   - 循环处理下一批

5. 可以在弹窗中查看实时进度和日志

## 注意事项

- 请确保当前页面为抖店商品管理页面
- 插件会自动处理，无需人工干预
- 每批处理间隔可在设置中调整
- 可以随时点击"停止"按钮暂停
